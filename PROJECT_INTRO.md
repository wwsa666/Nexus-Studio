# Neural Studio - 项目介绍

**Neural Studio** 是一款专为 ComfyUI 设计的现代化桌面客户端，旨在为 AI 绘画爱好者和专业人士提供一个优雅、高效且功能强大的创作环境。它通过直观的图形界面（GUI）简化了 ComfyUI 的复杂操作，同时保持了高度的可定制性和专业性。

## 核心功能亮点

### 1. 现代化用户界面 (Modern UI)
- **沉浸式设计**：采用深色主题（Dark Mode），融合了 Glassmorphism（毛玻璃）效果和流畅的微交互动画，提供极佳的视觉体验。
- **响应式布局**：界面布局灵活，左右分栏设计（控制面板 + 画布），支持自定义窗口大小并自动记忆上次状态（全屏/窗口位置）。

### 2. 强大的提示词编辑 (Prompt Engineering)
- **智能编辑**：提供正向（Positive）和负向（Negative）提示词专用输入区，支持语法高亮（已优化去除拼写错误红线）。
- **角色标签系统**：自动识别和展示提示词中的角色标签（Tags），支持**点击一键复制**，方便快速复用。
- **快捷操作**：支持右键菜单复制、清空等快捷操作。

### 3. 无缝 ComfyUI 集成
- **自动连接**：启动时自动尝试连接本地 ComfyUI 服务。
- **模型管理**：实时加载和切换 Checkpoints（大模型）、Samplers（采样器）和 Schedulers（调度器）。
- **参数调优**：
  - 支持 Width/Height 尺寸调节（256px - 2048px）。
  - Steps（步数）、CFG Scale（提示词相关性）、Batch Size（批量生成）等核心参数通过滑块精准控制。
  - Seed（随机种子）控制，支持 -1 随机或指定种子复现。

### 4. 实时生成与预览
- **实时进度**：生成过程中显示旋转加载动画、计时器（精确到 0.1s）和当前状态。
- **交互式画布**：
  - **缩放与平移**：支持鼠标滚轮缩放、拖拽平移查看图片细节。
  - **键盘导航**：支持使用键盘 `←` / `→` 方向键快速切换上一张/下一张图片。
- **历史记录**：侧边栏自动保存生成的图片历史，支持点击回看，且**重启软件后记录不丢失**（持久化存储）。

### 5. 文件与数据管理
- **智能保存**：支持一键保存图片到本地，或直接打开输出文件夹。
- **剪贴板集成**：支持一键复制图片或提示词到系统剪贴板。
- **状态记忆**：自动记忆窗口大小、位置以及最后使用的模型设置。

### 6. 技术栈 (Tech Stack)
本项目基于最前沿的现代 Web 技术构建，确保了高性能和跨平台兼容性：
- **Electron**: 跨平台桌面应用框架。
- **React 18**: 用于构建动态用户界面的 JavaScript 库。
- **TypeScript**: 提供类型安全，保证代码质量。
- **TailwindCSS**: 原子化 CSS 框架，实现快速且一致的样式开发。
- **Zustand**: 轻量级状态管理库，配合 `persist` 中间件实现数据持久化。
- **Vite**: 下一代前端构建工具，提供极速的开发体验。

---

## 快速开始

1. 确保本地已启动 ComfyUI 服务。
2. 运行 Neural Studio 客户端。
3. 在左侧面板选择模型、输入提示词。
4. 点击 "Generate" 开始创作！
